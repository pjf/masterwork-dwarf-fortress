Trading is very important in this mod. Each allied Race will send both caravans and Merchant Liaisons. You can order any item these races produce. Each race has specialisations as well as "crates". Crates are a custom item that can be unpacked into a mix of wanted items, for example a steel bar crate will produce 10 steel bars. An Armor-set crate will generate a helm, breastplate, greaves, 2 boots and a shield. These will ease trading.

Each race has unique, powerful items that you cannot produce yourself. You really wants these items in the late game to be able to repell the more difficult invaders. You should build corridors and protect the caravans well.
